# ⚽ Torneo de Fútbol App

Aplicación desarrollada en Flutter para gestionar torneos de fútbol.

## Funcionalidades

- Crear torneos
- Registrar equipos
- Generar fixture de partidos
- Visualizar tabla de posiciones
- Gestionar resultados

## Tecnologías utilizadas

- Flutter
- Dart
- Visual Studio Code
- Git
- GitHub
